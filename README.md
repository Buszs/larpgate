# Larpgate

Situs resmi **Larpgate** — tim pengembang game indie yang lahir dari sekumpulan teman kuliah di tahun 2026.

🔗 **Live site:** [buszs.github.io/larpgate](https://buszs.github.io/larpgate/)

## Tentang

Undefined Team adalah tim baru di industri game, tapi punya ambisi besar. Kami membuat game dengan sumber daya terbatas, dibekali kreativitas dan tekad untuk terus belajar dari setiap project.

> "Innovative games, unforgettable experiences."

## Pencapaian

**Looping Life: Dimensional Escape** — Juara 2 di 3 kategori (Njess, Quiver, Raveanne) pada Live2Dev Game Jam 2025 dengan tema *"Cost"*. Dibuat hanya dalam 2 hari, dan sekarang sedang dikembangkan menjadi rilis penuh.
